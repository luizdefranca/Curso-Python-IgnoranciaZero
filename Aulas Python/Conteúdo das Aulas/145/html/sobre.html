    • Projeto Ignorancia Zero

    O canal ignorância zero busca compartilhar gratuitamente conhecimento a todos os que estiveram dispostos a aprender e se desenvolver. O canal não tem nenhum certificado em especial, ou um profissional ministrando as aulas, apenas uma pessoa qualquer com disposição para divulgar aquilo que aprendeu. O objetivos, por enquanto, é ensinar lógica de programação (utilizando uma linguagem simples como python), para em seguida aprender a desenvolver jogos (computador, tablet, videogames) e aplicativos, esses ultimos para android e iOS principalmente.
